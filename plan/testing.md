# Testing Plan

## Tujuan Pengujian

Memastikan fitur berjalan sesuai kebutuhan dan aman digunakan.

## Jenis Pengujian

- Unit test.
- Integration test.
- End-to-end test.
- Manual testing.

## Skenario Utama

- Alur sukses.
- Input tidak valid.
- Kondisi data kosong.
- Kondisi error.

## Checklist Rilis

- [ ] Semua fitur utama sudah diuji.
- [ ] Tidak ada error kritis.
- [ ] UI responsif.
- [ ] Dokumentasi sudah diperbarui.
